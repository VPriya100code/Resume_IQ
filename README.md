## 🌟 Features

✨ **ATS Score Analyzer**
→ Get instant resume scoring like real hiring systems

🎯 **Resume vs Job Matching**
→ Compare your resume with job descriptions and identify gaps

🧠 **AI Resume Improver**
→ Rewrite weak points into strong, impactful statements

📊 **Skill Gap Detection**
→ Discover missing skills required for your dream role

📈 **Interview Prediction**
→ Estimate your chances of getting shortlisted

🛣️ **Skill Roadmap Generator**
→ Get a personalized learning path to improve

🎤 **Voice Feedback (Bonus)**
→ Listen to AI feedback in real-time

🌐 **Portfolio Generator**
→ Convert your resume into a personal portfolio website

## ⚡ How It Works

1️⃣ Upload your resume (PDF/TXT)
2️⃣ Paste a job description
3️⃣ Click **Analyze**
4️⃣ Get AI-powered insights instantly

## 🚀 Getting Started

```bash
# Clone the repo
git clone https://github.com/your-username/resumeiq-ai.git

# Navigate into project
cd resumeiq-ai

# Install dependencies
npm install

# Run the app
npm run dev
```
